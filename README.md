**NewsNest** is a mobile-first news publishing app built using Android Studio, Kotlin, and Jetpack Compose.  
It integrates with the WordPress REST API to fetch and post articles, providing a streamlined platform for writing and managing news content.

---

## Features

- Jetpack Compose-based modern UI  
- Integration with WordPress REST API  
- Fetch latest articles from a WordPress site  
- Post news articles directly to the WordPress backend  
- Custom interface tailored for writing and editing news content  
- Centralized publishing interface for future multi-platform sharing

---

## Multi-Platform Publishing

**Currently implemented:**

- Post to WordPress website  
- Post to in-app reader  

**Planned features:**

- Share to WhatsApp  
- Post to Instagram  
- Integration with other social and messaging platforms

---

## Tech Stack

- Kotlin  
- Android Studio  
- Jetpack Compose  
- WordPress REST API
